# 🩺 알려주는 건강  
일상 속 건강 실천을 돕는 통합 건강 정보 탐색 웹 서비스  

> "파편적인 건강 정보, 이제는 한눈에 쉽게!"  
> 사용자가 질병에 대한 정보부터 운동·식단·환경 요소까지 통합적으로 탐색하고, 실질적인 건강 관리로 이어지도록 돕는 헬스 케어 플랫폼입니다.  

---

## 🧬 주요 기능  

**1. 당뇨 테스트 (전문가용 / 일반인용)**  
- Kaggle 당뇨병 데이터를 기반으로 한 Logistic Regression 모델로 당뇨병 위험도 예측  
- 전문가용: 수치 기반 입력값 / 일반인용: 질문 기반 간편 테스트  
- 결과에 따라 위험도 구간 분류 및 맞춤 운동 추천  

**2. 운동 추천**  
- ExerciseDB API 연동  
- 위험도에 따라 운동명을 자동 매핑  
- GIF 이미지, 타겟 근육, 수행 방법 등 시각적 자료와 함께 제공  
- Google Translate API를 통해 한국어 번역 지원  

**3. 음식 성분 검색**  
- 식품의약품안전처 영양성분DB API 연동  
- 음식명 검색 시 주요 영양소(열량, 탄수화물, 단백질 등) 정보 실시간 조회  
- 테이블 형태로 가독성 있게 출력  

**4. 체육관 찾기**  
- 카카오맵 API 기반  
- 사용자 위치 중심 반경 2km 이내 헬스장 및 체육시설 검색  
- 지도 위 마커 표시 + 이름/주소/전화번호 정보 제공  

**5. 날씨 / 미세먼지 기반 운동 추천**  
- 기상청 + 공공 미세먼지 API 연동  
- 날씨 + 대기질 조합에 따라 실내/야외 운동 추천  
- 응답 실패 시 fallback 문구 처리  

**6. 질병 정보 제공**  
- 질병관리청 API 연동  
- 질환별 정의, 증상, 예방법, 생활습관 등 정보 제공  
- API 오류 시 예시 데이터 제공  

**7. 음식 효능 검색**  
- 건강식품 효능 API 연동  
- 질병명을 선택하면 해당 질환에 도움이 되는 음식 목록 제공  
- 성분 검색 기능과 연계 가능 (예정)  

**8. 게시판 (커뮤니티)**  
- 자유 게시글 작성 및 공유 기능 (Oracle DB 기반)  
- 카테고리 필터, 검색, 댓글, 이미지 첨부 등 제공  
- 작성자 본인만 수정/삭제 가능  

**9. 마이페이지**  
- 내가 쓴 글 / 좋아요 / 북마크한 글 관리  
- 사용자의 활동 기록을 한눈에 확인 가능
